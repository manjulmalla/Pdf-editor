ℹ️ How to Use

  1. Upload Template: Upload a PDF template with the original certificate layout
  2. Manual Editor: Click "Manual Editor" to add, drag, resize, and type text boxes directly on the PDF
  3. Form Fill: Use "Form Fill" for a traditional form-based approach with predefined fields
  4. Configure Coordinates: Set exact X, Y coordinates for each field (for form-based approach)
  5. Generate PDF: Click generate to overlay text on the original template and create a print-ready PDF
  6. Download/Print: Download the generated PDF or open it in print preview

Manual Editor Features

  • Add Text Box: Click "Add Text Box" to create a new editable field
  • Drag & Position: Drag text boxes to position them exactly where you want
  • Resize: Drag corners to resize text boxes
  • Edit Text: Double-click a text box to edit its content
  • English & Nepali: Type in both English and Nepali (Unicode)
  • Save Layout: Save field positions for reuse

📐 Coordinate System

For the form-based approach, you can configure exact coordinates:

  • X Coordinate: Horizontal position (left to right)
  • Y Coordinate: Vertical position (bottom to top)
  • Font Size: Size of the text in points
  • Origin: (0, 0) is at the bottom-left corner of the PDF

Tip: Use the Manual Editor for visual placement, or configure coordinates manually.
